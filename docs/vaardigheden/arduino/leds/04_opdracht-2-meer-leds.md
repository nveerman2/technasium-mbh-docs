# Opdracht 2: Meer LEDs

In deze opdracht ga je meer LEDs toevoegen aan het circuit en leer je hoe je ze onafhankelijk van elkaar kunt laten knipperen. Dit laat je zien hoe je meerdere outputs kunt aansturen met behulp van een enkele Arduino.

---

## Tinkercad Opdracht

**2.1** Bouw de schakeling na zoals getoond in de afbeelding.  
**2.2** Programmeer de Arduino met de code van de afbeelding om de LED te laten knipperen.
